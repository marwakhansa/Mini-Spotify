# Mini Spotify Project

## Overview

The Mini Spotify Project is a Java-based application with an attractive graphical user interface (GUI) that simulates basic features of a music streaming platform. It manages songs, users, and playlists using a structured database. This project is developed for academic purposes to understand application design and database integration.

## Technologies Used

* Java (Core Java)
* Java GUI (Swing / JavaFX)
* SQL Database
* JDBC

## Features

* Simple and user-friendly GUI
* Add, view, update, and delete songs
* User and playlist management
* Database-based data storage
* Demonstrates basic CRUD operations

## Learning Objectives

* Learn Java GUI development
* Understand database connectivity using JDBC
* Practice SQL queries in applications
* Apply basic software design concepts

## How to Run

1. Clone the repository
2. Open the project in a Java IDE (IntelliJ, Eclipse, or NetBeans)
3. Configure the database connection
4. Run the main Java file
