Mini-Spotify


It is a simple java based music player project built to simulate Spotify features.


It includes the following features:
-Add a song by browsing a from your device
-Delete a song
-Search a song
-See current song details
-Add a song to playlist
-Basic UI controls: Play, Pause, Stop, Next, Previous and Progress bar


How can you run it:

-Clone the repository
-Open `index.html` in a browser
-Enjoy the music
